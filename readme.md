                       GRYD YouTube Downloader • by deACTA
                     ───────────────────────────────────────

   [ Release ] GRYD v1.0
   [ Type    ] Desktop Python Application
   [ Lang    ] Python 3.8+ / PySide6 (Qt6)
   [ UI      ] Animated • Dark/Light Themes • Multilingual
   [ Engine  ] yt-dlp (auto-updated)
   [ Audio   ] MP3 192kbps via FFmpeg
   [ Video   ] MP4/WEBM up to 4K
   [ Font    ] JetBrains Mono

   ──[ Features ]──────────────────────────────────────────────────────────────
   • Paste URL → Get info → Choose format → Download
   • Auto FFmpeg detection & merge
   • Smooth button animations on hover and click SFX
   • Theme-aware logo & icons
   • English / Русский interface
   • Zero bloat. Pure performance.

   ──[ Requirements ]──────────────────────────────────────────────────────────
   • Python 3.8+
   • FFmpeg (in PATH)
   • pip install -r requirements.txt

   ──[ Install ]───────────────────────────────────────────────────────────────
   $ git clone https://github.com/deACTA/GRYD.git
   $ cd GRYD
   $ pip install -r requirements.txt
   $ python GRYD.py
